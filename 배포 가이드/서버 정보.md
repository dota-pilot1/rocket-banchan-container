## 주의사항

- 도메인은 `dxline-tallent.com` 하나만 쓴다. 서브도메인 분리 없음.
- BeautyBook 백엔드(4101)는 Nginx에서 끊되, 컨테이너/데이터는 유지.
- `API_KEY_SECRET`은 한 번 정하면 바꾸지 않는다 (저장된 OpenAI 키 암호화에 사용).
- EC2 메모리는 2GB라 Java 서비스를 동시에 두 개 띄우지 않는다. BeautyBook이 멈춰 있어야 EnglishAgentHub가 여유롭게 동작한다.
