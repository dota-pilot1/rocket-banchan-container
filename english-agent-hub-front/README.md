# english-agent-hub-front

Next.js 16 + React 19 + Tailwind v4 기반 프론트엔드.

## Dev

```bash
npm install
npm run dev   # http://localhost:4300
```

## Env

- `NEXT_PUBLIC_API_URL` — 백엔드 API URL (기본 `http://localhost:3301`)

## Ports

| 서비스 | 포트 |
| --- | --- |
| Front (Next.js) | 4300 |
| Backend (Spring Boot) | 3301 |
| Postgres (host) | 5436 |
